# String&StringBuffer&StringBuilder

  1. 可变性

    StringBuffer&StringBuilder继承AbstractStringBuilder，这个类提供了修改字符串的append方法；

    string可不变：

    public final class String implements java.io.Serializable, Comparable<String>, CharSequence {
    private final char value[];
	//...
}
    1
    2
    3
    4
    1. 保存字符串的数组被final修饰且私有，且String没有对外暴露修改这个字符串的方法；
    2. String类被final修饰，不能被继承，避免子类破坏；

  2. 线程安全

    String对象不可变，必然安全；

    StringBuffer对方法加同步锁，线程安全；

    StringBuilder非线程安全；

  3. 性能

    String每次改变都生成一个新的String对象， 改变对象引用，适合少量数据；

    StringBuffer&StringBuilder都对对象本身操作；
